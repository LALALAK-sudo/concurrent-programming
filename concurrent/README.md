# 说明
* 代码主要在 case_java8 子模块内，按章节划分，例如
* jmh_performance 子模块
    * 使用了 JMH，演示多线程提升性能的代码
* jmh_eliminate_locks 子模块
    * 使用了 JMH，演示锁消除
* jcstress_ordering 子模块
    * 使用了 jcstress，演示指令重排序
* case_java7
    * 演示与 java7 相关的代码
* case_tomcat 子模块
    * 演示自定义 tomcat
* case_monitor 子模块
    * 演示多种设计模式的综合案例
    * balking
    * 两阶段终止
